Note: Redmine is a full Ruby on Rails application that requires a persistent application server. Vercel is optimized for static sites and serverless functions, which may not be ideal for hosting Redmine.

Recommended hosting platforms:

  • Heroku - Easy Rails deployment with database add-ons
  • Railway - Modern platform with simple Rails support
  • Render - Free tier available for Rails apps
  • Fly.io - Supports Rails with persistent storage
  • DigitalOcean App Platform - Managed Rails hosting

üìù To run Redmine locally:

1. Clone the repository:

git clone https://github.com/ericson-j-santos/redmine.git

2. Install dependencies:

bundle install

3. Setup database:

bundle exec rails db:create db:migrate

4. Start the server:

bundle exec rails server

5. Visit: http://localhost:3000
